# Zero Fog

**Zero Fog** is a lightweight Minecraft resource pack that removes all fog, giving players a clear and unobstructed view without impacting performance.

## Features
- Removes all fog
- Clear visibility in all dimensions
- Lightweight and performance-friendly
- Compatible with most mods and shaders

## Installation
1. Download the resource pack
2. Place it into the `resourcepacks` folder
3. Enable it from Minecraft settings

## Compatibility
- Works on modern Minecraft versions
- Supports shaders and modded environments
